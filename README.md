# College Helpdesk Chatbot 🎓

An AI-powered college helpdesk chatbot built using Flask, HTML, CSS, JavaScript, and OpenAI API.

## Features
- FAQ support using JSON database
- AI fallback responses
- Modern responsive UI
- College-related queries support

## Tech Stack
- Python Flask
- HTML, CSS, JavaScript
- OpenAI API
- JSON Database

## Run Locally

```bash
pip install flask openai
python app.py
```

## Sample Questions
- Fees
- Admission
- Placement
- Hostel
- Contact
